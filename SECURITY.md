# Security

## Supported Versions

Security fixes target the current `7.x` line.

## Reporting

Do not open a public issue for a security problem.

Use GitHub Security Advisories:

<https://github.com/golded-dev/golded-app/security/advisories/new>

Include:

- What is affected
- How to reproduce it
- Impact
- Any logs or sample input needed to verify it

Keep private archive data out of reports unless it is truly required. Synthetic reproduction data is better.
